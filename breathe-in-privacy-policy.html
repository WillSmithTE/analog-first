Privacy Policy

Last updated: February 02, 2025

Overview

Breathe In is a Chrome extension that adds a subtle breathing widget to your browser. The extension is designed with privacy in mind and operates entirely offline.

Data Collection

The extension does not collect or transmit any data. There are no analytics, tracking, or external services used.

Data Storage

The extension stores only widget positions per website in Chrome's local storage. This data never leaves your browser and is only used to remember where you placed the widget on each site.

Permissions

The extension requests minimal permissions:

  • Storage: To save widget positions locally
  • ActiveTab: To place the widget on your current tab
  • Host permissions: Requested per-site to maintain widget across page refreshes

Open Source

This extension is open source software under the MIT License. You can review the source code to verify these privacy claims.
